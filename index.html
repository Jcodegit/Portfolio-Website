Arduino RFID badge reader

I built an RFID access control system from scratch using an Arduino and the RC522 RFID module. I soldered headers onto the RC522 to insert it into the breadboard, wired the components, programmed the authentication logic, and added visual feedback using an OLED screen along with red and green LEDs to indicate "Access Denied" and "Access Granted" states, based on the hardcoded UID number in the tags.
