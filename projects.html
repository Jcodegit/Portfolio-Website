Active Directory Homelab

Built a virtual enterprise environment using Windows Server to model real-world identity and access management. I configured Active Directory Domain Services, DNS, organizational units, and shared resources, then validated role-based access from a domain-joined client. I spend a lot of time in this virtual Windows network and am continuously growing it, and taking courses to expand my system administration knowledge.

System Information Dashboard (Python)

Built a Python utility that collects and displays key system metrics in a dashboard-style output, including OS details, uptime, CPU utilization, RAM usage, disk usage, active user, and network information. The script uses various modules to present practical host-level diagnostics for support and troubleshooting workflows. This project strengthened my ability to use Python to interact with system-level resources.

Arduino RFID Access Control System (C/C++)

Built a physical access-control prototype using Arduino and an RC522 RFID module with C/C++. The system validates card IDs and provides real-time visual feedback for authorization state: prompt to scan, access denied, and access granted. This was a passion project that fed my curiosity for embedded systems and strengthened my ability to use code to interact with hardware.
